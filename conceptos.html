¿QUE ES CSS3?



CSS es un lenguaje de diseño grafico que permite definir y crear la presentacion de un documneto estructurado escrito en leguaje de mercado. Es muy usado para establecer el diseño visual de los documentos web en interfaces de usuarios escritas en HTML.

¿Cómo aplicar estilos internos y externos?

Ahora vamos a ver las tres formas en las que podemos situar estas reglas de estilo para que el navegador lea su contenido y utilice sus especificaciones para darle forma al documento.

A continuacion, encontraremos información sobre los estilos en CSS3.

Ventajas y Desventajas De Css3:

ventajas:

1. Utiliza un código especializado para crear un proyecto en sí.

2. Tiene múltiples elementos, para fomentar el buen desarrollo.

3. Logra estilos y efectos visuales nuevos y mejores, que antes sólo eran posibles por medio de tecnologías adicionales.

Desventajas:

1. El uso de las tablas nos permitía crear diseños complejos de forma mucho más sencilla que utilizando CSS, aunque CSS3 está intentando facilitar dicho trabajo.

2. Existen limitaciones que CSS 2.x todavía no permite, por ejemplo, la alineación vertical de capas, las sombras, los bordes redondeados.
