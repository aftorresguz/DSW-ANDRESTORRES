¿Que es CSS3?

CSS es un lenguaje de diseño grafico que permite definir y crear la presentacion de
un documneto estructurado escrito en leguaje de mercado. Es muy usado para establecer
el diseño visual de los documentos web en interfaces de usuarios escritas en HTML.
